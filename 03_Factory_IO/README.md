# Factory I/O

En esta carpeta se almacenarán simulaciones de procesos industriales desarrolladas en Factory I/O e integradas con controladores PLC.

## Objetivos

- Representar procesos industriales virtuales.
- Integrar sensores y actuadores.
- Comunicar Factory I/O con CODESYS.
- Validar programas PLC en un entorno simulado.
- Analizar condiciones normales y fallas del proceso.

## Proyectos planificados

- Control de cinta transportadora.
- Clasificación de productos.
- Control de nivel de estanque.
- Sistema automático de almacenamiento.
- Línea de producción con sensores.
- Simulación de fallas operacionales.

## Evidencias

Cada proyecto podrá incluir:

- Archivo de simulación.
- Capturas de pantalla.
- Diagrama de funcionamiento.
- Tabla de entradas y salidas.
- Video demostrativo.
- Informe técnico.
