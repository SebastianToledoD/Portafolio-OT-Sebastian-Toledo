# Sistemas SCADA y HMI

En esta carpeta se almacenarán proyectos de supervisión, control y adquisición de datos para procesos industriales.

## Competencias que se desarrollarán

- Diseño de interfaces HMI.
- Supervisión de variables industriales.
- Gestión de alarmas.
- Registro de eventos.
- Visualización de tendencias.
- Históricos de variables.
- Administración de usuarios.
- Comunicación con PLC y servidores OPC UA.

## Proyectos planificados

- HMI para control de motor.
- Supervisión de una bomba industrial.
- Sistema SCADA para estanque de agua.
- Supervisión de cinta transportadora.
- Gestión de alarmas y eventos.
- Panel de tendencias e históricos.

## Herramientas consideradas

- Ignition SCADA.
- CODESYS Visualization.
- OPC UA.
- Bases de datos para históricos.
