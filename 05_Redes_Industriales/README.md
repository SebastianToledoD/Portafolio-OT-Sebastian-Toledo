# Redes Industriales

En esta carpeta se documentarán arquitecturas, topologías y comunicaciones utilizadas en entornos de automatización industrial y sistemas OT.

## Competencias que se desarrollarán

- Diseño de topologías industriales.
- Direccionamiento IP.
- Segmentación mediante VLAN.
- Routing y switching.
- Redes OT e IT.
- Zonas y conductos.
- Análisis de tráfico con Wireshark.
- Comunicación entre PLC, SCADA y servidores.
- Disponibilidad y redundancia de red.

## Protocolos considerados

- Modbus TCP.
- Modbus RTU.
- OPC UA.
- PROFINET.
- EtherNet/IP.
- MQTT.
- DNP3.

## Proyectos planificados

- Red industrial básica PLC-SCADA.
- Segmentación entre red IT y red OT.
- Diseño de una DMZ industrial.
- Captura y análisis de tráfico Modbus TCP.
- Arquitectura industrial segmentada en GNS3.
- Integración de servicios industriales virtualizados.
