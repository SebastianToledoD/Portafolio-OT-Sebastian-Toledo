# Módulo 00 — Índice y Metodología

## Propósito del módulo

Este módulo define la organización, metodología, convenciones y sistema de control documental de la Biblioteca Profesional de Ingeniería Industrial y OT.

Su función es mantener una estructura uniforme durante el desarrollo de los diferentes módulos, capítulos, rutas de aprendizaje y proyectos industriales.

Este módulo no contiene principalmente conocimientos técnicos de electricidad o automatización. Contiene las reglas utilizadas para estudiar, redactar, revisar, organizar y publicar el conocimiento técnico de la biblioteca.

---

## Objetivos

Al completar este módulo se habrá definido:

* El propósito general de la biblioteca.
* El alcance técnico del proyecto.
* El método de aprendizaje.
* Las convenciones de escritura y nomenclatura.
* La plantilla oficial de capítulos.
* Las rutas de aprendizaje.
* El mapa de conocimientos.
* El sistema de seguimiento del avance.
* Los criterios para considerar un tema comprendido.
* El sistema de control de versiones.

## Principios generales

La biblioteca se desarrollará siguiendo los siguientes principios:

1. Se trabajará un tema técnico por vez.
2. Cada tema comenzará con un diagnóstico de conocimientos previos.
3. La explicación avanzará desde una comprensión intuitiva hacia una comprensión técnica.
4. Los capítulos serán redactados principalmente por el autor.
5. Cada explicación será revisada antes de generar el capítulo final.
6. Los conceptos se relacionarán con aplicaciones industriales.
7. Las afirmaciones normativas deberán contar con referencias confiables.
8. Se diferenciarán claramente las normas, recomendaciones, buenas prácticas y ejemplos didácticos.
9. Cada capítulo tendrá control de versión y estado de dominio.
10. Los conocimientos se integrarán mediante proyectos industriales.

## Organización del conocimiento

La biblioteca utiliza tres formas complementarias de organización:

### Módulos

Agrupan el conocimiento según su disciplina técnica.

Ejemplos:

* Fundamentos eléctricos.
* Motores eléctricos.
* PLC.
* Redes industriales.
* Ciberseguridad OT.

### Rutas de aprendizaje

Establecen el orden práctico en que se estudian los temas para desarrollar una competencia o proyecto.

La primera ruta corresponde al diseño completo de un arranque directo para un motor trifásico.

### Proyectos

Integran conocimientos pertenecientes a distintos módulos.

Un proyecto puede contener:

* Cálculos.
* Selección de equipos.
* Diagramas eléctricos.
* Programas de PLC.
* Pantallas HMI.
* Arquitectura de red.
* Diagnóstico de fallas.
* Análisis de riesgos.
* Documentación técnica.

## Estados de los documentos

Cada documento podrá encontrarse en uno de los siguientes estados:

| Estado                 | Significado                                                    |
| ---------------------- | -------------------------------------------------------------- |
| Pendiente              | El documento todavía no ha sido iniciado.                      |
| En estudio             | El tema se encuentra en etapa de aprendizaje.                  |
| En redacción           | El autor está preparando su explicación.                       |
| En revisión            | El contenido está siendo corregido técnicamente.               |
| Borrador               | Existe una versión inicial utilizable.                         |
| Completado             | El capítulo cumple los criterios establecidos.                 |
| Requiere actualización | El contenido debe revisarse por cambios técnicos o normativos. |
